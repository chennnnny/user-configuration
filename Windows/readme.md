`setup_envriment.bat`

用于环境配置。

`_bashrc.bat`

通过 `setup_envriment.bat` 来注册 `_bashrc.bat`，用于构建 `windows` 平台的 `bash` 环境。

---

`chocolatey_install.bat`

用于安装 `chocolatey包管理器`。

---

`choco_software.bat`

用 `chocolatey包管理器` 安装常用的软件。

---

`scoop_software.bat`

用 `scoop包管理器` 安装常用的软件。

---

`capslock2ctrl.reg`

将 `capslock` 键换成 `ctrl`，`vimer` 必备。
